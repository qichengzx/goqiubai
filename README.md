## 糗百首页热门

带翻页查看糗百24小时热门帖子。

### 运行

```
$ go get github.com/qichengzx/goqiubai
$ cd $GOPATH/src/github.com/qichengzx/goqiubai
$ go run main.go 
//或者
$ go build main.go
$ ./main
```
